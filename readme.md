# Thyroid Disease Detection Using Machine Learning

This is a mini-project aimed at detecting thyroid dysfunction (hypothyroidism, hyperthyroidism, or normal) using machine learning models trained on clinical lab data.

## 🔍 Project Overview

- Predicts thyroid status based on patient lab results (TSH, T3, TT4, etc.)
- Uses the UCI Thyroid Disease Dataset
- Tabular data only — no image processing
- Lightweight model that can run locally
- Clean, user-friendly interface (web or desktop)


## ⚙ Tech Stack

- Python
- Jupyter Notebook
- Pandas, NumPy
- scikit-learn
- Flask or Tkinter (for UI)

#
## ✅ Features

- Early thyroid disease detection
- No image data — works only on lab parameters
- Easy-to-use interface
- Optional: PDF reports, graphs, multilingual UI, admin panel

## 📊 Dataset

- Source: [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/59/thyroid+disease)
- Files: `hypothyroid.data`, `allbp.data`
- Samples: 7,000+
